随机图片API调用 自动更换图片网址

发表于2022-08-24|更新于2022-08-26|技术
|阅读量:

一些随机图片 API 的服务网址，随机显示图片，刷新就会自动更换图片的调用链接。

缙哥哥博客

  • 随机动漫图片：https://api.dujin.org/pic/
  • 必应每日1366×768分辨率：https://api.dujin.org/bing/1366.php
  • 必应每日1920×1080分辨率：https://api.dujin.org/bing/1920.php
  • 必应每日1080×1920分辨率：https://api.dujin.org/bing/m.php

岁月小筑

  • 随机动漫图片：https://img.xjh.me/random_img.php
  • 随机返回源网址：https://img.xjh.me/random_img.php?return=302
  • 随机自然环境背景：https://img.xjh.me/random_img.php?type=bg&ctype=nature
  • 随机动漫人物背景：https://img.xjh.me/random_img.php?type=bg&ctype=acg

小歪 API

  • 随机动漫图片：https://api.ixiaowai.cn/api/api.php
  • 随机mc酱图片：https://api.ixiaowai.cn/mcapi/mcapi.php
  • 随机高清壁纸图片：https://api.ixiaowai.cn/gqapi/gqapi.php

樱花 API

  • 随机动漫图片：https://www.dmoe.cc/random.php

搏天 API

  • 自适应随机图片：https://api.btstu.cn/sjbz/zsy.php
  • 横版动漫图片：https://api.btstu.cn/sjbz/?lx=dongman
  • 横版美女图片：https://api.btstu.cn/sjbz/?lx=meizi
  • 横版动漫妹子：https://api.btstu.cn/sjbz/?lx=suiji
  • 竖版动漫图片：https://api.btstu.cn/sjbz/?lx=m_dongman
  • 竖版美女图片：https://api.btstu.cn/sjbz/?lx=m_meizi
  • 竖版动漫妹子：http://api.btstu.cn/sjbz/?m_lx=suiji

Lorem Picsum

  • Unsplash 指定尺寸随机图：https://picsum.photos/1920/1080
  • Unsplash 指定尺寸随机图：https://unsplash.it/1600/900
